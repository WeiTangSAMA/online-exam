# 在线考试系统 — 使用手册

> 本系统基于 Web 浏览器运行，无需安装任何客户端软件。打开浏览器访问系统地址即可使用。

---

## 一、登录与注册

### 1.1 打开系统

在浏览器地址栏输入前端地址：

```
http://localhost:5173
```

进入登录页面。

### 1.2 登录

| 账号 | 密码 | 角色 | 权限说明 |
|------|------|------|----------|
| `admin` | `123456` | 管理员 | 拥有全部权限，管理系统用户、题库、试卷、成绩 |
| `teacher` | `123456` | 教师 | 管理题库、组卷发布、查看成绩、数据分析 |
| `student1` | `123456` | 学生 | 参加考试、查看个人成绩与答题详情 |
| `student2` | `123456` | 学生 | 同上 |
| `student3` | `123456` | 学生 | 同上 |

在登录页输入用户名和密码，点击 **「登 录」** 按钮即可进入系统。

> 💡 登录页下方有**测试账号一键填充**按钮，点击即可自动填入账号密码。

### 1.3 注册新账号

1. 点击登录页上方的 **「注册」** 选项卡
2. 填写用户名（不可重复）、密码（至少 6 位）、姓名
3. 选择角色（学生 / 教师）
4. 点击 **「注 册」**
5. 注册成功后自动切回登录页，使用新账号登录

> ⚠️ 管理员账号目前仅能通过数据库或由管理员手动创建，不支持自主注册。

### 1.4 退出登录

点击页面右上角头像区域，在下拉菜单中选择 **「退出登录」**。

---

## 二、系统首页（Dashboard）

登录后首先看到的是首页仪表盘，展示以下内容：

- **欢迎信息**：显示当前登录用户姓名和日期
- **数据概览**：题目总数、试卷总数、可参加考试数、成绩记录数（根据角色不同显示不同统计）
- **快捷入口**：点击卡片直接跳转到对应功能模块
- **系统介绍**：当前角色与技术栈说明

## 三、教师 / 管理员功能

> 以下功能需要使用 `admin` 或 `teacher` 账号登录。

---

### 3.1 题库管理

**菜单路径**：左侧导航 → **题库管理**

#### 3.1.1 查看题目列表

- 页面以表格形式展示所有题目
- 支持按 **题型**（单选 / 多选 / 判断）、**分类**、**关键词** 筛选
- 支持分页浏览（每页 10/20/50 条）

#### 3.1.2 新增题目

1. 点击右上角 **「新增题目」** 按钮
2. 在弹出的对话框中填写：
   - **题型**：选择单选题、多选题或判断题
   - **分类**：选择所属分类（如 Java基础、数据库等）
   - **题目内容**：输入题干文字
   - **选项设置**（单选/多选题）：添加至少 2 个选项（如 A. xxx、B. xxx）
   - **正确答案**：选择或填写正确选项
   - **分值**：设置该题分值（默认 5 分）
   - **答案解析**（选填）：填写题目解析说明
3. 点击 **「确定」** 保存

#### 3.1.3 编辑题目

点击表格中某道题目的 **「编辑」** 按钮，修改后点击确定保存。

#### 3.1.4 删除题目

点击 **「删除」** 按钮，二次确认后删除。

> ⚠️ 删除操作不可恢复，请谨慎操作。

#### 3.1.5 分类管理

点击 **「分类管理」** 按钮，可以：
- 在输入框中输入分类名称，点击添加
- 点击已有分类右侧的删除按钮移除分类

---

### 3.2 试卷管理

**菜单路径**：左侧导航 → **试卷管理**

#### 3.2.1 查看试卷列表

- 表格展示所有试卷，包含标题、总分、时长、及格分、发布状态
- 可按状态筛选（草稿 / 已发布）

#### 3.2.2 新建试卷

1. 点击 **「新增试卷」** 按钮
2. 填写试卷信息：
   - **试卷标题**：如「Java基础测试卷」
   - **考试时长**：单位分钟（如 60 分钟）
   - **及格分数**：如 60 分
   - **状态**：草稿（暂不发布）或立即发布
3. 点击 **「确定」** 创建试卷

#### 3.2.3 组卷（给试卷添加题目）

1. 在试卷列表中点击目标试卷的 **「组卷」** 按钮
2. 弹出组卷窗口，顶部显示已选题目数和合计分值
3. 使用筛选条件（题型 / 分类 / 关键词）查找题目
4. **勾选**要加入试卷的题目（支持跨页多选）
5. 点击 **「保存组卷」**

> 💡 已绑定过的题目会自动勾选显示，重新保存即可更新。

#### 3.2.4 发布试卷

- 点击试卷的 **「发布」** 按钮
- 发布后学生即可在「在线考试」页面看到并参加考试
- 二次确认后立即生效

#### 3.2.5 编辑 / 删除试卷

- **编辑**：修改试卷标题、时长、及格分等基础信息
- **删除**：删除试卷及其所有组卷关联（二次确认）

---

### 3.3 成绩管理

**菜单路径**：左侧导航 → **成绩管理**

#### 3.3.1 全部成绩

- 以表格展示所有学生的考试成绩
- 显示学生姓名、试卷名称、得分/总分、及格状态、提交时间
- 支持按 **试卷筛选**
- 点击 **「查看详情」** 可以查看该学生每道题的作答情况

#### 3.3.2 成绩排行

- 点击 **「成绩排行」** 切换到排行视图
- 按分数从高到低排列
- 可按试卷筛选排行

#### 3.3.3 查看答题详情

点击某条成绩记录的 **「查看详情」** 按钮：

- 显示每道题的：题型、题干、你的答案、正确答案、是否正确、得分
- 错题会用红色标记，正确答案用绿色标记
- 如果题目有解析，会在底部显示

---

### 3.4 数据分析

**菜单路径**：左侧导航 → **数据分析**

展示以下统计图表：

| 统计项 | 说明 |
|--------|------|
| **参考人数** | 总共参考人数 |
| **及格率** | 及格人数占比 |
| **平均分** | 所有成绩的平均分 |
| **最高/最低分** | 最高分和最低分 |
| **成绩分布柱状图** | 各分数段（0-59、60-69、70-79、80-89、90-100）的人数分布 |
| **及格率饼图** | 及格与不及格人数占比 |
| **TOP 10 排行榜** | 成绩最高的 10 条记录 |

可通过顶部下拉框按试卷筛选统计数据。

## 四、学生功能

> 以下功能使用 `student1`、`student2`、`student3` 账号登录。

---

### 4.1 参加考试

**菜单路径**：左侧导航 → **在线考试**

#### 4.1.1 查看可参加的考试

- 以卡片形式展示所有**已发布**的试卷
- 每张卡片显示试卷标题、考试时长、总分、及格分

#### 4.1.2 开始考试

1. 点击试卷卡片上的 **「开始考试」** 按钮
2. 二次确认后进入答题界面
3. 系统自动开始 **倒计时**

#### 4.1.3 答题界面说明

答题界面分为三个区域：

```
┌──────────────────────────────────────────────┐
│  📝 Java基础测试卷          ⏱️ 58:42       │  ← 顶部：考试名称 + 倒计时
├─────────┬────────────────────────────────────┤
│         │                                    │
│  答题卡  │     第 3 题 / 共 10 题    5分     │
│         │                                    │
│ [1][2]  │     以下哪个是Java的基本数据类型？  │
│ [3] ←当前│                                    │
│ [4][5]  │     ○ A. String                    │
│ [6][7]  │     ○ B. Integer                   │
│ [8][9]  │     ● C. int  ← 你的选择           │
│ [10]    │     ○ D. Boolean                   │
│         │                                    │
│         │     [上一题]    [下一题 →]          │
│         │                                    │
│ [交卷]   │                                    │  ← 底部：翻页 + 交卷
├─────────┴────────────────────────────────────┤
```

- **左侧答题卡**：显示所有题号，灰底=未答、黑底=已答、边框高亮=当前题
- **中间答题区**：显示题目内容、选项、作答区域
- **底部操作栏**：上一题 / 下一题 / 交卷

#### 4.1.4 作答方式

| 题型 | 操作方式 |
|------|----------|
| **单选题** | 点击选项前面的圆圈，选中一个 |
| **多选题** | 勾选多个选项前面的方框 |
| **判断题** | 点击「✓ 正确」或「✗ 错误」 |

#### 4.1.5 交卷

两种方式：
- **手动交卷**：点击底部 **「交卷」** 按钮（最后一题也有交卷按钮），二次确认后提交
- **自动交卷**：倒计时归零后系统自动提交

> 💡 答题过程中系统会**自动保存**你的答案到浏览器本地缓存，即使不小心刷新页面也不会丢失。但最终成绩以点击交卷时提交的内容为准。

#### 4.1.6 查看成绩

交卷后弹出成绩结果弹窗，显示你的得分和是否通过。点击 **「查看成绩」** 跳转到成绩页面。

---

### 4.2 查看成绩

**菜单路径**：左侧导航 → **成绩管理**

- 以表格展示你的所有考试成绩
- 显示试卷名称、得分/总分、是否及格、提交时间
- 点击 **「查看详情」** 可查看每道题的：
  - 你的答案与正确答案对比
  - 每题得分与对错标记
  - 题目解析（如果有）

## 五、常见问题

### Q1：登录后看不到某些菜单？
**A：** 不同角色看到的功能不同。学生只能看到「在线考试」和「成绩管理」，教师和管理员能看到所有功能。

### Q2：考试过程中不小心关了浏览器怎么办？
**A：** 重新打开浏览器登录系统，再次进入同一考试即可。你的答案已自动保存在浏览器本地。但注意考试倒计时不会暂停。

### Q3：为什么交卷后成绩为 0？
**A：** 请确认是否选择了答案再交卷。未作答的题目得 0 分。可在成绩页点击「查看详情」核对每题的作答情况。

### Q4：如何修改个人密码？
**A：** 当前版本暂不支持在线修改密码，需联系管理员在数据库中修改。

### Q5：可以重新参加同一次考试吗？
**A：** 目前每个学生对每张试卷只能参加一次。如需重考，请联系教师删除该成绩记录。
